HTML

  • Gives the content structure, like the frame of a house
  • Elements are case sensitive but declarations are not
  • Attributes are always in the beginning

CSS

  • There are 3 types: internal, inline and external. The latter is the most common.
  • CSS rules have selectors and declarations.
  • External CSS won't work unless you link the location.

GIT

  • Helps you manage each version of your project
  • You can push and pull between remote branches
  • Allows you to code without disrupting the main code

JavaScript

  • Helps users to interact with the web app
  • Ends with a semicolon; Strings are enclosed in quotations
  • Use arrays, loops and functions for JS
